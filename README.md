# CocosAndriodNativeCode
基于cocoscreator3.8.x构建andriod项目后，生成的原生公共目录native/engine/common、原生平台目录native/engine/android

建议：
Android Studio Android Studio Otter 2 Feature Drop | 2025.2.2 Patch (used Java 21.0.8)
Android Gradle Plugin Version 8.0.2 (cocoscreator 3.8.1 默认)
Gradle Version 8.5  (Java 21.0.8 兼容的最低 Gradle 版本为 8.5)
NDK Version 21.4.7075529
SDK Version
    platforms android-33
    build-tools 30.0.3 (cocoscreator 3.8.1 默认)
